## Assignment Description

This repository contains the Cisco Packet Tracer files completed for the Computer Networks routing assignment. The tasks cover basic router configuration, static routing, and dynamic routing using RIP and EIGRP protocols.

## Files Included

### task01.pkt

Basic router setup including:

* Hostname configuration
* Console password configuration
* Console login authentication

### task02.pkt

Static routing configuration including:

* IP address assignment
* Static route configuration
* Connectivity testing between networks

### task03_rip.pkt

Dynamic routing using RIP:

* RIP protocol configuration
* End-to-end connectivity verification

### task03_eigrp.pkt

Dynamic routing using EIGRP:

* EIGRP protocol configuration
* Connectivity verification between all connected networks

## Testing

All configurations were tested to ensure successful communication between devices and networks. Verification was performed using:

■ Ping tests between devices
■ Routing table inspection using router commands
■ End-to-end connectivity checks
